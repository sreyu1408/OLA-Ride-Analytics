# OLA-Ride-Analytics
End-to-end data analytics project using SQL, Power BI, and Streamlit for ride booking insights.

## Tools Used

* SQL (MySQL)
* Power BI
* Python (Streamlit, Pandas)
* Data Cleaning

## Key Features

* Ride trend analysis
* Revenue insights
* Cancellation analysis
* Customer rating analysis
* Interactive Streamlit dashboard

##  How to Run Streamlit App

```bash
python -m streamlit run app.py
```
